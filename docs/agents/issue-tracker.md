# Issue tracker: GitHub

Issues and PRDs for this repo live in GitHub Issues at https://github.com/ChengenHsieh0225/kafka-sigma-engine.

## Conventions

- Use the `gh` CLI to create, read, and update issues.
- Apply triage labels as defined in `triage-labels.md`.
- PRDs are posted as GitHub Issues with a `PRD:` prefix in the title.

## When a skill says "publish to the issue tracker"

Run: `gh issue create --title "..." --body "..." --label "..."`

## When a skill says "fetch the relevant ticket"

Run: `gh issue view <number>` or `gh issue list`.
